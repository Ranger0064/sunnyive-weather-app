# sunnyive-weather-app
A minimalist, ad-free weather application built with Svelte and Capacitor. Focused on extreme privacy with zero tracking and no cookies, providing a clean and reliable weather experience for Android and Web.    一款基於 Svelte 和 Capacitor 建構的極簡主義、無廣告天氣應用程式。它專注於極致隱私保護，零追蹤、無 Cookie，為 Android 和網頁使用者提供簡潔可靠的天氣體驗。
